# Data Cleaning Portfolio

## Projects

### 1. Cafe Sales Data Cleaning
- **Dataset:** 10,000 rows of cafe sales transactions (Kaggle)
- **Problems fixed:** 6,151 missing values, 1,172 price mismatches, 
  460 bad dates, inconsistent text
- **Result:** 9,540 clean rows (95.4% retained)
- **Tools:** Python, Pandas, NumPy
- [View notebook](cafe_sales_cleaning.ipynb)
- [Download clean file](cafe_sales_clean.csv)
